# RC4 library for MacOS

An RC4 encryption dynamic library.

## Description

This was witten to explore creating a shared library under Linux and MacOS. The RC4 encryption was chosen as it is handy to have, and fun to write. This is meant as an educational experience.

## WARNING

This code is experimental and has not been validated against any other implimentation. Use at your own risk.
